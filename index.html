Pod Classic

Carica MP3 dal computer o dal telefono. I file restano locali: non vengono inviati online.

Usa la ghiera per scorrere, il centro per selezionare, MENU per tornare indietro. Su iPhone/Android serve un gesto dell’utente per far partire l’audio: premi il tasto centrale o Play.

Nota: per provarla correttamente usa un piccolo server locale o GitHub Pages, non aprire direttamente index.html con file://.
